# PC5AtmStudies/OHP_Studies
=========================


* Author : Sylvie Dagoret-Campagne
* affiliation : LAL/IN2P3/CNRS
* Creation date : May 2017


This directory has been created for a meeting
on stardice and the atmosphere at LPNHE
by the end of may 2017.

These files intend to show how one can calculate horizontal transmittance at OHP.

Some data file have been extracted from LibRadTran.
Other are obtained from HITRAN.

The Hitran interface is used.
